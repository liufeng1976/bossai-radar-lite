# BossAI Radar Lite 与商业 Pro 版边界

**中文** · [English](LITE_VS_PRO_EN.md)

Lite 的目标是让用户验证“公开证据 → 机会评分 → CEO 行动”的核心方法；Pro 的目标是支持企业持续经营、团队协作和客户交付。

| 能力 | Lite 源码开放非商业版 | Pro 商业版 |
|---|---|---|
| Reddit / Hacker News / GitHub | 支持公开接口 | 支持并增加稳定代理、配额和监控 |
| X / Product Hunt / 商业数据库 | 不包含 | 按授权接入 |
| Upwork / 外包需求 / 收入案例 Feed | 不包含 | 支持合规 Feed 与人工核验流程 |
| 本地 SQLite | 支持 | PostgreSQL、多实例与备份策略 |
| 单管理员 API Key | 支持 | 用户、团队、角色、租户隔离和审计 |
| 确定性机会评分 | 支持 | 支持行业权重、客户自定义规则和版本审计 |
| 可选 DeepSeek 分析 | 支持 | 多模型路由、成本预算、降级和质量评估 |
| CEO 日报 | 支持 | 日报、周报、专题报告和客户品牌报告 |
| 飞书 / Slack / 邮件 / 企业微信 | 不包含 | 支持通知、审批和分发策略 |
| CRM 与销售线索分配 | 不包含 | 支持 |
| 行业模板 | 基础通用分类 | 电商、跨境、SaaS、内容、招聘等商业模板 |
| 演示数据 | 明确标记的合成数据 | 可配置企业 Demo 与销售演示环境 |
| 白标和品牌替换 | 不允许 | 按商业合同授权 |
| 企业内部营收用途 | 必须另行授权 | 包含在对应商业授权中 |
| 托管、SLA 和技术支持 | 社区式、无 SLA | 可选托管、升级、培训和 SLA |
| 商业再分发 | 不允许 | 仅在取得再分发授权后允许 |

## 产品升级路径

1. 使用 Lite 验证安装、采集、评分和报告体验；
2. 明确企业需要的数据源、行业和团队人数；
3. 选择内部使用、客户交付、白标或再分发授权；
4. 迁移至 Pro 数据库和权限体系；
5. 配置企业数据源、模型、推送与报告模板；
6. 完成上线验收和持续支持。
