## 安全

- [ ] `.env`、`data/`、SQLite、日志未进入发布包
- [ ] 默认管理员密钥仅用于本机，不用于公网
- [ ] 公网部署说明要求 HTTPS 和强密钥
- [ ] 演示入口可通过环境变量关闭
- [ ] 所有采集器有超时、限量和失败隔离
- [ ] 公开线索接口不能读取、导出、修改或删除线索
- [ ] 管理员线索接口由强 `RADAR_ADMIN_API_KEY` 保护
- [ ] 跟进队列、话术、日报和日历接口全部经过管理员鉴权
- [ ] WON / LOST 不进入活跃跟进队列
- [ ] 管理提示语言与客户话术语言分离
- [ ] 不把原始 IP、Cookie、设备指纹或管理员密钥写入线索数据库
- [ ] 线索去重、限流、蜜罐、同意确认和删除级联通过测试
- [ ] 多币种报价分别统计，不把不同币种直接相加
